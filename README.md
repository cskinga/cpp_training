# Feladatok elkészítése

1. File / New / Project...
2. Válassz template-t!
3. Így nevezd el a projektet: {forrás}\_{fejezet száma}\_{feladat sorszáma}
4. A Solution mezõben állítsd be, hogy Add to solution!

Példaként létrehoztam a feladatgy_I_1.cpp projektet.

Ahhoz, hogy tetszõlegesen tudd futtatni a projekteket a kijelölt fájl alapján:
1. Nyisd meg a Solution Explorer-t!
2. Kattints a Properties gombra vagy üsd le az Alt+Enter billentyûkombinációt!
3. A Common Properties / Startup Project felületen válaszd a Current selection lehetõséget!

# Verziókövetés

## Klónozás

1. Hozz létre egy új mappát a gépeden cplusplus_training néven!
2. Nyisd meg böngészõben: https://github.com/cskinga/cpp_training
3. A zöld Code gombra kattintva megjelennek a klónozási lehetõségek.
4. Másold ki a https útvonalat!
5. Nyisd meg a Visual Studio 2019-et!
6. Válaszd a Clone repository lehetõséget!
7. A Repository location mezõbe illeszd be a kimásolt útvonalat!
8. A Path mezõben add meg - az 1. lépésben létrehozott - cplusplus_training mappa útvonalát!
9. Kattints a Clone gombra!
10. Ha még nem jelentkeztél be a Visual Studio-ba, akkor kérni fogja a github-os adataidat.

## Commit

1. Ha nincs megnyitva, nyisd meg a View / Git changes ablakot.
2. Nyomd meg a Pull gombot!
3. Írd le, milyen módosításokat végeztél!
4. Válaszd ki a Commit All helyett a Commit All and Push lehetõséget!
